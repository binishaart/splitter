# Simple Expense Splitter 💰

A simple web app to split a bill among multiple people.

## Features
- Easy input for total bill and number of people
- Validates input
- Shows amount per person

## Run Steps
1. Clone the repo:
```bash
git clone <your-repo-link>
cd expense_splitter
Install dependencies:

pip install -r requirements.txt


Run the app locally:

streamlit run app.py
